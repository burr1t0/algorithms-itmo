### Пример оценки

**Задача:** Реализовать функцию подсчета слов в строке

Решение:

```c
void countWords(const char* text) {
    if (!text) return 0;

    Vector* keys = createVector(100);  // O(1), точно знаем что строки до 100 символов
    Vector* counts = createVector(sizeof(int));  // O(1)

    char word[100];  // O(1), так как 100 бесконечно мало (<<) с возможным N (размером входа)
    word[0] = '\0';  // O(1)

    for (int i = 0; text[i] != '\0'; i++) {  // O(K)
        char c = text[i];  // O(K) * O(1)

        if (!isspace((unsigned char)c)) {  // O(K) * O(1)
            strcat(word, c);  // O(K) * O(1)
        } else {
            long index = findVectorItem(keys, word, CmpStrings);  // O(K) * O(N)
            if (index == -1) {
                appendVectorItem(keys, &word);  // O(K) * O(1) - считаем добавление как O(1)
                int count = 1;
                appendVectorItem(counts, &count);  // O(K) * O(1) - считаем добавление как O(1)
            } else {
                int count = *(int*)getVectorItem(counts, index) + 1;  // O(K) * O(1)
                setVectorItem(counts, index, &count);  // O(K) * O(1)
            }
            word[0] = '\0';
        }
    }

    /*
    Оценка сверху - O(K * N)
    Точная оценка - Θ(K * N)
    Оценка снизу - Ω(K * N), в любом случае пройдем все символы и будем искать во всех словах

    Реализация плохая, можно сделать за O(K) или O(N), где N = K * 100
    */
}
```

В этом примере я указал плохую оценку, в вашем случае можете сразу оптимизировать и писать уже оптимизированную оценку. Ваши улучшения будут видны через PR.
